# Крестики-нолики

Классическая игра "Крестики-нолики" для двух игроков, на поле 3х3.

## Описание игры

### Правила игры
- Играют два игрока: первый ставит "X", второй ставит "O".
- Игроки по очереди ставят свои символы в свободные клетки поля 3×3.
- Побеждает тот, кто первым выстроит три своих символа в линию по горизонталь, вертикали или диагонали.
- Если все клетки заполнены, но победитель не выявлен, объявляется ничья, а игра завершается.

### Логика победы/поражения
Победа наступает, когда один из игроков заполняет три клетки одной линии:
- 3 горизонтальные линии
- 3 вертикальные линии
- 2 диагональные линии

Поражения как такового нет, игра завершается либо победой одного из игроков, либо ничьей.

### Особенности реализации
- Визуальная обратная связь при наведении на клетки
- Автоматическое переключение ходов
- Кнопка перезапуска игры

## Игровая механика

### Как определяется победитель?
После каждого хода программа проверяет все 8 возможных выигрышных комбинаций:
- 3 горизонтали (строки)
- 3 вертикали (столбцы)
- 2 диагонали

Если все три клетки в какой-либо комбинации содержат одинаковый символ и не пусты, фиксируется победа.

### Как обрабатываются клики?
1. При клике на клетку определяется её индекс (data-index)
2. Проверяется: свободна ли клетка и активна ли игра
3. Если условия выполнены — в клетку ставится символ текущего игрока
4. Обновляется массив игрового поля
5. Запускается проверка результата (победа/ничья/продолжение игры)

### Как работает перезапуск?
При нажатии кнопки "Начать заново":
- Очищается игровое поле (все клетки становятся пустыми)
- Сбрасывается массив board
- Текущий игрок становится "X"
- Игра становится активной
- Обновляется текст статуса

## Структура данных

### Массив board
```javascript
board = ["", "", "", "", "", "", "", "", ""]
```

Массив из 9 элементов, где каждый элемент хранит свой символ, а также соответствует клетке игрового поля:

Индекс 0 — левая верхняя клетка
Индекс 1 — верхняя средняя
Индекс 2 — верхняя правая
Индекс 3 — средняя левая
Индекс 4 — центральная
Индекс 5 — средняя правая
Индекс 6 — нижняя левая
Индекс 7 — нижняя средняя
Индекс 8 — нижняя правая

Пустая строка " " означает, что клетка свободна. Символы "X" или "O" означают, что клетка занята

## Условия победы

Победа засчитывается, когда 3 символа одного ирока встали в линию (по горизонтали, вертикали или диагонали)

## Состояние игры

Состояние игры хранится в трёх переменных:

- currentPlayer - текущий игрок
- gameActive - флаг активности игры
- statusText - текст статуса игры

## Алгоритмы

##### Алгоритм проверки победы

Перебор всех выигрышных комбинаций и проверка на совпадение символов в каждой тройке.

##### Алгоритм определения ничьи

Проверка наличия пустых клеток в массиве board. Если пустых нет и победы нет - ничья.

##### Алгоритм смены игрока
После каждого успешного хода `currentPlayer` переключается между "X" и "O".
